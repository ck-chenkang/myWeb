# 开发积累

## 不要一个类名把所有的样式写完，可以多带几个类名

比如有绿、大、线，一旦携带这个类名，就有相应的样式变化。对应 css 里的代码如下：

```
<style type="text/css">
    .lv {
        color: green;
    }
    .da {
        font-size: 30px;
    }
    .xian {
        text-decoration: underline;
    }
</style>
```

然后让每个标签去选取自己想要用的类选择器：

```
<p class="lv da">段落1</p>
<p class="lv xian">段落2</p>
<p class="da xian">段落3</p>
```
